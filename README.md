Student Management System 

A simple console-based Student Management System built using Python, Pandas, and CSV file handling.
This project allows users to:
 -  Add student records
 -  View all students
 -  Search students by ID
 -  Update student details
 -  Delete student records
 -  Find the topper
 -  Calculate average marks
 -  Display students sorted by marks

Features
 - Add Student
 - View All Students
 - Search Student by ID
 - Update Student Details
 - Delete Student
 - Find Topper
 - Find Average Marks
 - Sort Students by Marks
